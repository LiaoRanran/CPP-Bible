## 一、C++ 历史与标准（第 1–3 章）

- **第 1 章 C 与 C with Classes：C++ 的基因**
  - 1.1 C 语言的诞生与成功（Bell 实验室、Unix、可移植性）
  - 1.2 C 的局限：强类型缺失、抽象能力不足、大型工程痛点
  - 1.3 1979 Bjarne 的 "C with Classes"：类、派生类、强类型检查
  - 1.4 1983 更名 C++、虚函数、重载、引用、new/delete 引入
  - 1.5 1985 《The C++ Programming Language》与 CFront 编译器
  - 1.6 标准化前夜：多厂商方言割据（CFront / Borland / Microsoft）
- **第 2 章 标准演进治理与时间线**
  - 2.1 WG21 组织结构、提案（paper）、邮件列表、会议
  - 2.2 三年周期与"列车模型"（C++20 起）
  - 2.3 提案生命周期：EWGI → LEWG → LWG → PL22.16
  - 2.4 特性分类：语言核心 / 库 / TS（技术规格）
  - 2.5 标准文档阅读法：N 编号、章节寻址、footnote 与 note 的区别
- **第 3 章 各标准版本特性详解（C++98 → C++26）**
  - 3.1 C++98：模板、异常、RTTI、STL 入标准、命名空间
  - 3.2 C++03：值初始化修复、export 模板（后废除）
  - 3.3 C++11（革命）：auto/decltype、lambda、右值引用、移动语义、变参模板、
         constexpr、nullptr、范围 for、强类型枚举、智能指针、thread、初始化列表、
         static_assert、uniform initialization → **逐特性讲动机/提案/工业影响/代码**
  - 3.4 C++14：泛型 lambda、返回类型推导、变量模板、泛型 lamba 捕获、std::make_unique
  - 3.5 C++17：结构化绑定、if/switch 初始化、折叠表达式、std::optional/variant/any、
         std::string_view、文件系统 TS、并行算法、inline 变量、if constexpr 落地库
  - 3.6 C++20（现代分水岭）：Concepts、Ranges、Coroutines、Modules、<=>、
         std::span、std::format、std::jthread、constexpr 虚、calendar/time、features
  - 3.7 C++23：std::expected、std::print、explicit object param（deducing this）、
         std::mdspan、std::flat_map/set、multidimensional subscript、#ifdef 替代品、
         std::generator、stacktrace、native handles
  - 3.8 C++26（已确定内容，标注实验性）：反射 P2996、契约 P2900、发送者-接收者 P2300、
         线性代数 P1673、std::execution、静态反射、格式化增强、模式匹配（方向）
  - 3.9 版本间破坏性变更与弃用策略（deprecation → removal）
